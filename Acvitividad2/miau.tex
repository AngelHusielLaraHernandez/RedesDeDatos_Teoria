\section{Comunicacion}
Se define como la transferencia de datos a traves de un medio de transmision entre dispositivos a traves de un medio de transmision este puede ser fisico o inalambrico, para exista la comunicacion se necesita de un mensaje, de emisior, el canal por el cual viaja el mensaje y el receptor.

\section{Redes de comunicacion}

Se define como un conjunto de equipos/dispositivos interconectados mediante conexiones físicas o inalámbricas con el proposito de permitir la transmisión y el intercambio de información y recursos, los cuales tienen que tienen protocolos.


\section{Transmision de datos}

Se entiende por transmision de datos al movimiento de infromacion codificada de un dispositivo digital a otro a través de canales de red, ya sean cableados o inalámbricos.
 
\section{Datos}

Los datos es informacion traducida a un formato eficiente para el movimiento o procesamiento de la misma, la cual puede ser almacenada y transmitida.

\section{Nodo}

Un nodo es un punto de conecion que peude recibir, enviar, crear o alamcenar datos, este mismo puede reconocer, procesar y transmitir informacion a otro nodo de la red.

\section{Ancho de banda}

Cantidad de informacion que puede fluir a traves de una conexion de red en un periodo determinado de tiempo, se mide en bits por segundo (bps) y es un factor clave para determinar la velocidad de una conexion de red.

\section{Dispositivo}

Es un componente de hardware que es capaz de enviar, recibir, crear o almacenar datos dentro de una infraestructura de red.

\section{Protocolos}

Los protocolos son conjuntos de reglas, normas y procedimientos estrictos que gobiernan la comunicación entre dispositivos

\section{Medios de transmision}
Los medios de transmisión son los canales físicos o inalámbricos a través de los cuales se transmiten los datos en una red de comunicación. 

\section{Cliente/Servidor}
Los clientes son dispositivos que solicitan servicios o recursos.

El servidor se encargan de proporcionar los servicios a las estaciones de trabajo conectadas a la red (clientes). 

\section{Direccion IP}

Una dirección IP es un número único asignado a cada dispositivo de red en una red IP (Protocolo de Internet). 
Esta dirección se utiliza para identificar y localizar dispositivos en la red, permitiendo la comunicación entre ellos. 


\section{Servicio de red}
Un servicio de red es una función o recurso proporcionado por un sistema en una red que permite a los usuarios o aplicaciones acceder a información o funcionalidades de una red.

\section{Topologia}

Las topologías hacen referencia a la forma como una red se estructura de forma física; es decir, la manera en que cada componente de una red se conecta con los demás. Hay diversas topologías, cada una con ventajas y desventajas determinantes para el rendimiento de la red.


\section{Conexion}
Una conexión se refiere al enlace establecido entre dos o más dispositivos en una red de comunicación, permitiendo la transferencia de datos y la comunicación entre ellos. Las conexiones pueden ser físicas, como cables Ethernet, o inalámbricas, como Wi-Fi o Bluetooth.
